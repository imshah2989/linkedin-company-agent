# LinkedIn Company Agent API — Backend

This is the FastAPI-based backend for the LinkedIn Company Agent, hosted using the Gradio SDK on Hugging Face Spaces.

## 🚀 Deployment Info
- **Framework**: FastAPI
- **SDK**: Gradio (for hybrid hosting)
- **Port**: 7860
- **Main Entry Point**: `app.py`

## 🛠 Tech Stack
- FastAPI & Uvicorn
- Cerebras AI (Message Generation)
- Serper.dev (X-ray Search)
- Google Sheets (Storage)
